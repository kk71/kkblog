kkblog
======
### 一个基于django1.5和python3的简易博客系统 ###
### a simple blog system based on django1.5 and python3 ###

Features 功能
-------------
* 基本博客支持(博文、分类、日期归档、标签)
* 评论、留言支持(包括评论的引用，博主回复时自动发送邮件)
* 背景音乐随机播放(且可设置是否自动播放)
* 博文编辑支持html
* 支持代码高亮
* 支持简易图片上传


third-part library 第三方库
---------------------------
* Mako
* grappelli
* simplejson
